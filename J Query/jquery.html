Welcome to J Query Learning Webpage.


To get the features of J Query there are two methods

In first Method We need to download the JS file from "jquery.com" inorder to get library file. (in this case the file will be offline)

In Second Method we use Content Delivery Network (CDN) where we will not be downloading any file but if we want to access this we need to connect to internet.

The process for Method 2 Content Delivery Network (CDN) is
first Go to jquery.com website and then Under "Download jQuery" section in website scroll down then we can see "Using jQuery with a CDN" and under that(in 3rd para) we can observe there is a hyperlink "https://code.jquery.com" and after clicking we can see all the CDN's available and from then we can select uncompressed one then we can get the code part and we can directly paste the code from that website to our code.
