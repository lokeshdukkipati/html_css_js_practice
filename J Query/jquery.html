now let us know about J Query Selectors.

jQuery selectors allow you to select and manipulate HTML element(s).

jQuery selectors are used to "find" (or select) HTML elements based on their name, id, classes, types, attributes, values of attributes and much more. It's based on the existing CSS Selectors, and in addition, it has some own custom selectors.

now let us know further about selectors.

$("#first,#second").action()
if we want to apply our animation for more than one "id" we make use of above syntax.
$("#first,p").action()
if we want to apply our animation for multiple things i.e; first one in which we assigned id as "first" and second one with all the paragraphs we make use of above syntax. [in case we want to add to more we can add using the same syntax]
$("div > p").action() or $("div p").action()
if we want to apply our animation to all the paragraphs which are in div tag we can use anyone from the above syntax [in the above example we can use them for any others also like which are in parent tag an child tag]
$("p:first").action()
if we want to apply our animation to the "first" paragraph only then we can make use of above syntax. [p can be changes to any other tag].
$("p:even").action()
if we want to apply our animation to the "even" paragraphs only then we can make use of above syntax. [p can be changes to any other tag like div tags, heading tags... etc;].
$("p:odd").action()
if we want to apply our animation to the "odd" paragraphs only then we can make use of above syntax. [p can be changes to any other tag like div tags, heading tags... etc;].
