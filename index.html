now let us know about "Regular Expressions". Regular expressions (RegEx) in JavaScript are patterns used to match character combinations in strings. They're commonly used for searching, validating, extracting, or replacing text.

  • using regular expressions we can set any conditions in the inputs from the users like in usernames.
  • If we want some specific pattern in our username then RegEx are very much helpful.
  • If we use the symbol "^" and mention any characters in that then those characters can not be included at that particular sequence. e.g; if we use [^7-9] then in that particular position the numbers 7-9 cannot be present and it shows invalid. The symbol ^ should be placed inside square braces [] then only it works.
  • Whenver we place the symbol "^" outside the sqaure braces then it indicates the starting of the expression. e.g; if we use ^[7-9] the starting of the expression must lie between 7-9.

The below are some of the popular keywords that are used in RegEx

\d
To match any digit (equal to [0-0]).
\w
To match any word character (a-z,A-Z,0-9 & _).
\s
To match whitespace character (eg - spaces and tabs).
\t
To match a tab only.
^[]
Denotes start the string.
$
Denotes end of the string.
dot (.)
When we use dot in our RegEx we can place any thing in that particular place. e.g; if our RegEx is /^a.b$/ any inputs like acb, adb or even a1b will be valid.
